# Python Fundamentals - École 42

Maîtrise des fondamentaux Python et introduction à la programmation fonctionnelle.

## Exercices

**Ex00 - Hello World**  
Manipulation de types natifs Python (list, tuple, set, dict).

**Ex01 - Format Time**  
Formatage de dates et timestamps avec la bibliothèque datetime.

**Ex02 - Find Type**  
Détection dynamique de types avec introspection Python.

**Ex03 - NULL Not Found**  
Gestion des valeurs nulles et types spéciaux (None, NaN, inf).

**Ex04 - What Is**  
Vérification de parité avec assertions et gestion d'arguments.

**Ex05 - Building**  
Analyse statistique de chaînes : comptage de caractères par catégorie.

**Ex06 - Filter String**  
Implémentation de filter() et expressions lambda pour filtrage avancé.

**Ex07 - SOS**  
Encodage de texte en morse avec dictionnaires et validation d'entrée.

**Ex08 - Loading**  
Création d'une barre de progression personnalisée (type tqdm).

## Stack
Python • sys • datetime

- Programmation fonctionnelle
