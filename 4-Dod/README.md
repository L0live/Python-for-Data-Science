# Data Oriented Design (Dod) - École 42

Concepts avancés de programmation Python : fonctions d'ordre supérieur, décorateurs, closures et dataclasses.

## Exercices

**Ex00 - Statistics**  
Implémentation de fonctions statistiques (moyenne, médiane, quartiles, écart-type, variance).  
Utilisation de `*args` et `**kwargs` pour une interface flexible.

**Ex01 - In & Out**  
Closures et fonctions d'ordre supérieur.  
Création de compteurs avec état privé persistant via `nonlocal`.

**Ex02 - Call Limit**  
Décorateur limitant le nombre d'appels d'une fonction.  
Implémentation d'un décorateur paramétré avec factory pattern.

**Ex03 - New Student**  
Utilisation des dataclasses pour créer des objets immuables.  
Génération automatique d'identifiants avec `field()` et `default_factory`.  
Initialisation post-création avec `__post_init__`.

## Stack
Python • Decorators • Dataclasses • Closures

## Compétences
- Fonctions variadiques (`*args`, `**kwargs`)
- Fonctions d'ordre supérieur
- Closures et `nonlocal`
- Décorateurs simples et paramétrés
- Factory pattern
- Dataclasses et fields
- Génération d'identifiants uniques
- Méthodes post-initialisation
