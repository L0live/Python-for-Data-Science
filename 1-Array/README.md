# NumPy Array Exercises - École 42

Manipulation de matrices et traitement d'images avec NumPy.

## Exercices

**Ex00 - Give my BMI**  
Calcul vectorisé d'indices de masse corporelle.

**Ex01 - Array Slicing**  
Extraction de sous-matrices par indexing avancé.

**Ex02 - Load Image**  
Chargement et affichage d'images en arrays NumPy.

**Ex03 - Zoom**  
Agrandissement d'images par duplication matricielle.

**Ex04 - Rotation**  
Rotation d'images 90° via opérations matricielles.

## Stack
Python • NumPy • Matplotlib

- Arrays multidimensionnels
- Broadcasting
