# Python for Data Science - École 42

Formation complète Python orientée Data Science : des fondamentaux aux analyses avancées.

## Modules

### 0 - Starting
Maîtrise des fondamentaux Python : types, structures de données, gestion d'arguments, programmation fonctionnelle et manipulation de chaînes.

### 1 - Array
Manipulation de matrices et traitement d'images avec NumPy : broadcasting, slicing avancé et opérations matricielles.

### 2 - DataTable
Analyse exploratoire avec Pandas et visualisation de données démographiques mondiales du dataset Gapminder.

### 3 - OOP
Programmation orientée objet avancée : classes abstraites, héritage multiple, surcharge d'opérateurs et résolution du problème du diamant.

### 4 - Dod (Data Oriented Design)
Concepts avancés de Data Oriented Design : fonctions d'ordre supérieur, closures, décorateurs paramétrés et dataclasses.

## Stack
Python • NumPy • Pandas • Matplotlib • Seaborn

## Compétences
- Programmation Python avancée
- Programmation orientée objet (OOP)
- Décorateurs et closures
- Fonctions d'ordre supérieur
- Manipulation de données structurées
- Visualisation scientifique
- Analyse statistique
- Traitement d'images
