# Object-Oriented Programming - École 42

Maîtrise de la programmation orientée objet en Python : classes abstraites, héritage et concepts avancés.

## Exercices

**Ex00 - Abstract Classes**  
Introduction aux classes abstraites avec ABC (Abstract Base Classes).  
Implémentation de `Character` et `Stark` avec méthodes abstraites.

**Ex01 - Inheritance & Static Methods**  
Héritage de classes avec `Baratheon` et `Lannister`.  
Utilisation de méthodes statiques comme factory patterns.  
Surcharge des méthodes `__str__` et `__repr__`.

**Ex02 - Multiple Inheritance (Diamond Problem)**  
Résolution du problème du diamant avec la classe `King`.  
Héritage multiple de `Baratheon` et `Lannister`.  
Implémentation de getters/setters pour les attributs physiques.

**Ex03 - Operator Overloading**  
Surcharge d'opérateurs arithmétiques (`__add__`, `__mul__`, `__sub__`, `__truediv__`).  
Création d'une calculatrice appliquant des opérations sur des listes.

**Ex04 - Static Methods for Vectors**  
Implémentation de méthodes statiques pour opérations vectorielles.  
Calcul de produit scalaire, addition et soustraction de vecteurs.

## Stack
Python • ABC (Abstract Base Classes) • OOP

## Compétences
- Classes abstraites et interfaces
- Héritage simple et multiple
- Method Resolution Order (MRO)
- Méthodes spéciales (`__str__`, `__repr__`)
- Surcharge d'opérateurs
- Méthodes statiques et factory patterns
- Getters et setters
- Résolution du problème du diamant
